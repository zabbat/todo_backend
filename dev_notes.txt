# Todo Spring Boot Backend

Backend for the Android Todo app.
- Kotlin
- Spring Boot
- Firebase Auth (with JWT)
- Dev DB: H2
- Prod DB: PostgreSQL (via Docker)

## Dev Setup Locally
./gradlew bootRunDev
## Dockers
dockerDevUp/dockerDevDown

## Production
./gradlew bootRunProd

## Dockers
dockerProdUp/dockerProdDown


## Tests
./gradlew test
